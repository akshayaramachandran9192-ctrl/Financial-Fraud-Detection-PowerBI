# Financial Fraud Detection Analysis 📊🏦

An interactive and dynamic Power BI end-to-end data analytics dashboard designed to analyze bank transactions, evaluate categorical channels, and track fraudulent patterns efficiently.

![Dashboard Screenshot](./Screenshot%2026-06-20%131438.png)

## 📌 Core Business Insights & Metrics
* **Total Dataset Rows Analyzed:** 100,000 Historical Transaction Records
* **Total Transactions Count:** 100.00K
* **Total Financial Volume Transacted:** ₹17.36 Billion (bn)
* **Highest Risk Channels Evaluated:** 
  * **TRANSFER Channel:** Dominating with nearly **₹7.4bn** transaction volume framework.
  * **CASH_OUT Channel:** Secondary heavy data metric volume at **₹5.9bn**.

## 🛠️ Tech Stack & Skills Demonstrated
* **Tool used:** Microsoft Power BI Desktop
* **Data Formatting:** Applied standard local Currency notations (`₹`) and dynamic numeric display auto-scaling configurations.
* **UI/UX Best Practices:** Implemented modern gridless card structures, intuitive typography hierarchy (Segoe UI family), and clean horizontal visual alignment patterns.
* **Interactive Slicers:** Enabled immediate multi-dimensional filtering across transaction types and fraud safety classifications.
